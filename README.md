Cricket Calculator Pro

Cricket Calculator Pro is a mobile application built with Ionic Angular and Capacitor.

Features

- Net Run Rate Calculator
- DLS Calculator
- Strike Rate Calculator
- Batting Average Calculator
- Bowling Average Calculator
- Bowling Economy Calculator
- Bowling Strike Rate Calculator
- Boundary Percentage Calculator
- Required Run Rate Calculator
- Run Rate Per Over Calculator

Technology Stack

- Ionic Angular
- Capacitor
- TypeScript
- Android

Ad Integration

- Google AdMob Interstitial Ads
